About this API

First off, I'm not some cape-wearing, superhero fanboy. Nor am I a fan of Marvel's Cinematic Universe. I'm just a dude trying very hard to be a developer by making cool side projects, albeit useless.

I recently stumbled upon MCU's Avengers series. And I kinda liked everything that baldheaded, ridge-chined, purple-ass motherfucker Thanos said. (The dude's a legit philosopher.) So I made an API dedicated to his badassery. This is Thanos API, or TAPI. Some Titanian-level wisdom right here.

Request and Response Examples

How to get started? Simple.

  • Make a GET request to https://thanosipsum.onrender.com/api
  • Use the endpoints to sculpt the output however you like.

Example:

  • Need three random quotes? thanosipsum.onrender.com/api/quotes/3
  • Want all quotes from Infinity War? thanosipsum.onrender.com/quotes/infinity

The response format is JSON, because even Thanos doesn't mess with XML.

For instance, the following command (using faithful old cURL) would get you two whole paragraphs of uncut Thanos quotes from the movie Endgame:

curl -X GET https://thanosipsum.com/api/quotes/2/endgame
